# 🧭 AI 길잡이 — 목적 기반 생성형 AI 활용 추천 웹 서비스

> 지금 **하려는 작업**을 입력하면, 상황에 가장 알맞은 **생성형 AI 모델**과
> **프롬프트 전략**을 추천해 주는 인터랙티브 웹 서비스입니다.

경남과학고등학교 **학생주도 프로젝트형 봉사활동** 결과물입니다.
생성형 AI가 학업 현장에 빠르게 도입되고 있지만, 목적에 맞는 AI 선택과
효과적인 프롬프트 작성법을 모른 채 활용하는 학생이 많습니다. 이 서비스는
사용자의 작업 목적을 입력받아 **어떤 AI를 어떻게 써야 하는지** 안내합니다.

---

## ✨ 주요 기능

- **자연어 입력 기반 추천** — "환경 보호 주제로 800자 에세이 쓰고 싶어"처럼
  하려는 일을 문장으로 적으면 작업 유형을 자동 분류합니다.
- **카테고리 선택** — 글쓰기, 코딩, 요약, 자료조사 등 9가지 작업 종류를
  버튼으로 바로 고를 수 있습니다.
- **AI 모델 추천** — 작업에 알맞은 AI를 1~3순위로 추천하고, 각 모델의 강점·
  무료 사용 범위·바로가기 링크를 함께 제공합니다. (Claude, ChatGPT, Gemini,
  Perplexity, NotebookLM, GitHub Copilot, 이미지 생성 AI, Wolfram Alpha 등
  2026년 최신 기준 8종)
- **프롬프트 전략 + 복사용 템플릿** — 작업별 프롬프트 작성 요령과, 바로
  복사해서 쓸 수 있는 프롬프트 템플릿을 제공합니다.
- **공통 프롬프트 기본기** — 어떤 AI를 쓰든 통하는 6가지 원칙을 안내합니다.

## 🗂 프로젝트 구조

```
.
├── index.html        # 메인 페이지 (화면 구조)
├── css/
│   └── style.css     # 디자인·스타일
└── js/
    ├── data.js       # 추천 데이터셋 (AI 모델 + 작업별 프롬프트 전략)
    └── app.js        # 추천 로직 (입력 분석 → 유형 분류 → 추천 출력)
```

이 서비스는 **별도의 서버나 빌드 과정이 필요 없는** 순수 정적 웹사이트입니다.
HTML/CSS/JavaScript만으로 동작하므로 누구나 쉽게 열어보고 수정할 수 있습니다.

## 🚀 실행 방법

### 1) 바로 열기
`index.html` 파일을 웹 브라우저로 열기만 하면 됩니다.

### 2) 로컬 서버로 실행 (권장)
```bash
# Python이 설치되어 있다면
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

### 3) GitHub Pages 배포
1. 이 저장소의 **Settings → Pages** 로 이동
2. **Source** 를 `Deploy from a branch` 로 설정
3. 브랜치를 선택하고 `/ (root)` 폴더 지정 후 저장
4. 잠시 후 발급되는 주소로 접속하면 누구나 사용할 수 있습니다.

## 🛠 추천 데이터 수정·확장하기

새로운 AI 모델이 나오거나 정보를 바꾸고 싶다면 **`js/data.js` 파일만**
수정하면 됩니다. (코드 변경 불필요)

- **AI 모델 추가/수정** → `AI_MODELS` 객체 편집
- **작업 유형·키워드·추천 모델·프롬프트 전략 수정** → `TASK_TYPES` 배열 편집
- **공통 프롬프트 팁 수정** → `GENERAL_TIPS` 배열 편집

각 항목에는 자세한 한국어 주석이 달려 있어 누구나 쉽게 고칠 수 있습니다.
